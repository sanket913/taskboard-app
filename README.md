# 🚀 TaskBoard App

A modern full-stack Task Management Web Application built using React and ASP.NET Core.

---

## 📌 Features

### 👤 User Features
- User Signup & Login (JWT Authentication)
- Create, Update, Delete Tasks
- Filter tasks (ToDo, In Progress, Done)
- Responsive and modern UI

### 🛠️ Admin Features
- View all users
- Change user roles (User ↔ Admin)
- Delete users
- View & delete all tasks

---

## 🧑‍💻 Tech Stack

### Frontend
- React.js
- React Router
- CSS (Custom Styling)

### Backend
- ASP.NET Core Web API
- Entity Framework Core
- SQLite Database
- JWT Authentication
